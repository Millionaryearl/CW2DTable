Cw2dtable

CW2DTable is a self-defined UICollectionViewController to simulate commercial statistic tables.

View project on GitHub

CW2DTable

It's inspired by the stupid requirement to display commercial statistic table. User could customize the data source in CW2DTableVC.m file - refreshSrchRlt method. A specific demonstration of the data structure could be found there.

Demo

Usage

To run the example project, clone the repo, and run pod install from the Example directory first.

Installation

CW2DTable is available through CocoaPods. To install it, simply add the following line to your Podfile:

pod "CW2DTable"
